# aradin
阿拉丁基础开发框架
以SpringCloud及SpringCloud Alibaba为基础做上层通用功能包扩展，  
  * 规范依赖
  * 降低集成复杂度    
  * 扩充部分必要的能力     
  * 解决部分新旧组件交替淘汰时发生的兼容性问题  

# v0.0.1 基础能力  
  * 基础结构
  * 健康检查
  * caffeine缓存集成
  * lts集成
  * redis集成
  * redisson集成
  * shiro集成
  * swagger集成
  * velocity兼容
  * 一般性web防护
  * zookeeper监听集成
  * 基于zookeeper版本同步
  * 基于zookeeper集群注册
